Multiply By Four Callback:-

The Function mulBy4() will multiply 4 to Each Element of the array which is passed as Argument in Higher Order Function


            function mulBy2(array) {
                var result = [];
                for (let i = 0; i < array.length; i++) {
                  var y = array[i] * 4;
                  result.push(y);
                }
                return result;
              }

              function add2(array) {

Higher Order Function For Above CallBack

The function manipulateArray is a Higher Order Function That Returns result Depending upon which Callback is passed into it while Invoking it


              function manipulateArray(array,callback) {
                let result=callback(array)
                return result;
              }
